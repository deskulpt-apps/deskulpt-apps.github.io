Struct SerError 

Source 
pub struct SerError(Error);
Expand description

Serializable wrapper around anyhow::Error.

This implements Serialize with the Debug representation of the error. Any error that can be converted into an anyhow::Error can be converted into this error type, meaning that error propagation with ? works in the same way as with anyhow::Error.
