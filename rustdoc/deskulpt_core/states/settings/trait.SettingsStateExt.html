Source

fn manage_settings(&self)

Initialize state management for the settings.

This will load the settings from the persistence directory and initialize the shortcuts. If any step fails, it will fall back to a state that preserves as much persisted data as possible.

Source

fn get_settings(&self) -> RwLockReadGuard<'_, Settings>

Get an immutable reference to the settings.

The returned reference is behind a lock guard, which should be dropped as soon as possible to minimize critical section.

Source

fn on_theme_change<F>(&self, hook: F)
where F: Fn(&Theme, &Theme) + Send + Sync + 'static,

Register a hook that will be triggered on theme change.

The two arguments are respectively the old and new themes.

Source

fn on_shortcut_change<F>(&self, hook: F)
where F: Fn(&ShortcutKey, Option<&String>, Option<&String>) + Send + Sync + 'static,

Register a hook that will be triggered on shortcut change.

The first argument is the shortcut key. The second and third arguments are respectively the old and new shortcuts. None means that no shortcut was/is assigned for that key.
