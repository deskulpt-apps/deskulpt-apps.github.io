fn get_widget_catalog(&self) -> RwLockReadGuard<'_, WidgetCatalog>

Get an immutable reference to the widget catalog.

The returned reference is behind a lock guard, which should be dropped as soon as possible to minimize critical section.

fn reload_widgets_all(&self) -> Result<()>
where Self: Sized,

Reload all widgets.

This function loads a new widget catalog from the widgets directory and replaces the existing catalog. It then syncs the settings with the updated catalog. If any step fails, an error is returned.

Source

fn render_widget(&self, id: String) -> Result<()>

Render a specific widget by its ID.

If the widget does not exist in the catalog, an error is returned. Otherwise, a render task is sent to the render worker. If this fails, an error is returned as well. This function does not wait for the rendering to complete.

Source

fn render_widgets_all(&self) -> Result<()>

Render all widgets in the catalog.

This function sends render tasks for all widgets in the catalog to the render worker. If any task fails to be sent, an error is returned containing all individual errors. This function does not wait for the rendering to complete.
