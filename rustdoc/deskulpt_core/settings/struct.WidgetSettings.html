pub struct WidgetSettings {
    pub x: i32,
    pub y: i32,
    pub width: u32,
    pub height: u32,
    pub opacity: u8,
}
Expand description

Per-widget settings.

Different from widget configurations, these are independent of the widget configuration files and are managed internally by the application.

Fields§

§x: i32

The leftmost x-coordinate in pixels.

§y: i32

The topmost y-coordinate in pixels.

§width: u32

The width in pixels.

§height: u32

The height in pixels.

§opacity: u8

The opacity in percentage.
